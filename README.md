Projeto de Otimização de Portfólio

INTEGRANTES:

Este projeto implementa quatro abordagens para resolver o problema clássico de otimização de portfólio (variação do problema da mochila 0/1). O objetivo é selecionar os ativos que maximizam o retorno esperado sem ultrapassar o capital disponível.

É necessário um aplicativo para rodar o codigo em python!
O codigo contém recursos de:
Greedy
Recursivo Puro
Memoization
Bottom-Up
